# 2. 整体框架

```
.
├── .gitignore
├── README.md
├── bin
│   └── start  ---------------------- 整个脚手架的入口文件(脚本)，无后缀
├── command
│   └── init.js  -------------------- 项目构建模板
│   └── page.js  -------------------- 新建一个页面
│   └── component.js  --------------- 新建一个组件页面
├── package.json
└── templates.json  ----------------- 存放模板信息
```
